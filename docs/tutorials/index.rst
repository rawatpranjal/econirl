Overview
--------

**Rust Bus Replication**
   Replicate the classic Rust (1987) bus engine replacement estimation.
   Learn the complete workflow from environment setup to counterfactual analysis.

**Rust (1987) Replication API**
   Programmatic replication of Rust's original tables with the replication module.
   Includes Table II, Table IV, and Table V generation with LaTeX export.

**Simulated Data Workflow**
   Understand parameter recovery using Monte Carlo simulation.
   Set known "true" parameters, generate data, estimate, and compare.

**Real Data Example**
   Apply econirl to actual empirical data.
   Complete workflow from data loading to interpretation.
