\section{Setup}
%% ======================================================================

Consider a stationary infinite-horizon MDP with finite state space $\mathcal{S}$, finite action space $\mathcal{A}$, discount factor $\beta \in (0,1)$, transition kernel $P : \mathcal{S} \times \mathcal{A} \to \Delta(\mathcal{S})$, and unknown reward $r : \mathcal{S} \times \mathcal{A} \to \mathbb{R}$. Additive T1EV shocks yield the soft Bellman recursion.
\begin{align}
Q^*(s,a) &= r(s,a) + \beta \sum_{s'} P(s' \mid s,a)\, V^*(s'), \label{eq:bellman} \\
V^*(s) &= \log \sum_{a'} \exp Q^*(s,a'), \label{eq:value} \\
\pi^*(a \mid s) &= \exp\bigl(Q^*(s,a) - V^*(s)\bigr). \label{eq:policy}
\end{align}
The advantage $A^*(s,a) = Q^*(s,a) - V^*(s) = \log \pi^*(a \mid s)$ is identified from CCPs. The reward $r$ is not identified in levels: any $\tilde{r}(s,a) = r(s,a) + \lambda(s)$ generates identical CCPs \citep{magnac2002}. The set of all such $\tilde{r}$ is the identified set \citep{kalouptsidi2021}.

\section{Five Approaches}
%% ======================================================================

\subsection{Reduced-form Q}

The reduced-form approach estimates $Q_\omega(s,a)$ by logit MLE.
\begin{equation}\label{eq:rf}
\hat\omega = \arg\max_\omega \sum_{i,t} \log \frac{\exp Q_\omega(s_{it},a_{it})}{\sum_{a'}\exp Q_\omega(s_{it},a')}.
\end{equation}
At the population optimum, $Q_{\hat\omega}(s,a) = Q^*(s,a) + c(s)$ for some action-independent $c$, so the backed-out ``reward'' is $A^*(s,a)$, not $r(s,a)$. No Bellman equation is imposed and no transition data are used.

\subsection{AIRL}

The AIRL estimator of \citet{fu2018} uses the following discriminator decomposition.
\begin{equation}\label{eq:decomp}
f_\theta(s,a,s') = g_\theta(s,a) + \beta\, h_\phi(s') - h_\phi(s).
\end{equation}
Here $g_\theta$ approximates the reward and $\beta h(s') - h(s)$ is potential-based shaping \citep{ng1999}. At the GAN optimum the discriminator score equals the advantage, $f^* = A^*$ \citep[Appendix A.4]{fu2018}, which implies the following expression for the recovered reward network.
\begin{equation}\label{eq:gstar}
g^*(s,a) = r(s,a) - \delta(s) + \beta \sum_{s'} P(s' \mid s,a)\,\delta(s'),
\end{equation}
where $\delta(s) \equiv V^*(s) - h^*(s)$ is the shaping residual. Unless $\delta \equiv 0$, the recovered $g^*$ is a potential-based perturbation of $r$ that generates the same policy in the original MDP but can produce different behavior under changed dynamics.

Fu et al.\ address this through a state-only restriction: if $g_\theta$ depends only on $s$ (not $a$) and the dynamics satisfy a decomposability condition, then $g^*(s) = r(s) + \text{const}$, recovering a disentangled reward that transfers across dynamics. This is what enables their sim-to-real experiments. However, the state-only restriction requires $r(s,a) = r(s)$, which fails whenever rewards depend on actions (e.g., paying vs.\ waiting).

\subsubsection{LSW anchor identification}

\citet{lee2026} (henceforth LSW) take a different approach. They allow action-dependent rewards and impose two anchors instead. First, the exit payoff is normalized to zero, so $g(s,\text{exit}) = 0$ for all $s$. Second, the shaping function is pinned at the absorbing state, so $h(s_{\mathrm{abs}}) = 0$, where exit deterministically transitions to $s_{\mathrm{abs}}$.

\begin{lemma}[Non-identifiability]\label{lem:nonid}
If $(g,h)$ and $(\bar{g},\bar{h})$ produce the same $f$ for all $(s,a,s')$, then $\bar{h} = h + \Phi$ and $\bar{g}(s,a) = g(s,a) + \Phi(s) - \beta\Phi(s')$ for some $\Phi : \mathcal{S} \to \mathbb{R}$.
\end{lemma}
\begin{proof}
Rearrange $g + \beta h(s') - h(s) = \bar{g} + \beta\bar{h}(s') - \bar{h}(s)$ and define $\Phi = \bar{h} - h$. \qed
\end{proof}

\begin{theorem}[Anchor identification]\label{thm:anchor}
Under the two anchors, if $(g,h)$ and $(\bar{g},\bar{h})$ realize the same $f$, then $g = \bar{g}$ and $h = \bar{h}$.
\end{theorem}
\begin{proof}
By Lemma~\ref{lem:nonid}, $\bar{g}(s,\text{exit}) = 0 + \Phi(s) - \beta\Phi(s_{\mathrm{abs}})$. The exit anchor forces $\Phi(s) = \beta\Phi(s_{\mathrm{abs}})$ for all $s$. The absorbing anchor forces $\Phi(s_{\mathrm{abs}}) = 0$. Hence $\Phi \equiv 0$. \qed
\end{proof}

\begin{theorem}[Recovery of true reward]\label{thm:recovery}
If the true pair $(r^*, V^*)$ satisfies $r^*(s,\text{exit}) = 0$ and $V^*(s_{\mathrm{abs}}) = 0$, then any AIRL pair $(g,h)$ realizing $A^*$ under the same anchors satisfies $g = r^*$ and $h = V^*$.
\end{theorem}
\begin{proof}
Because $(g,h)$ realizes $A^*$, for every realized transition $(s,a,s')$ we have
\[
g(s,a) + \beta h(s') - h(s) = r^*(s,a) + \beta V^*(s') - V^*(s).
\]
Define $\Phi = h - V^*$. Then
\[
g(s,a) - r^*(s,a) = \Phi(s) - \beta\Phi(s').
\]
For the exit action, $s' = s_{\mathrm{abs}}$ deterministically, and both the true reward and the recovered reward satisfy the exit anchor. Hence
\[
0 = g(s,\text{exit}) - r^*(s,\text{exit}) = \Phi(s) - \beta\Phi(s_{\mathrm{abs}}).
\]
The absorbing-state anchor applied to both $h$ and $V^*$ gives $\Phi(s_{\mathrm{abs}}) = 0$, so $\Phi(s) = 0$ for all $s$. Therefore $h = V^*$, and substituting back gives $g = r^*$. \qed
\end{proof}

The ERM framework of \citet{kang2025} uses the same anchor principle but solves a single non-adversarial optimization instead of a GAN.

The LSW model estimates segment-specific reward and policy functions for $K$ latent consumer types. Each user belongs to segment $z \in \{1, \ldots, K\}$, and the objective is to recover the segment-specific reward $r^*(s, a, z)$ and policy $\pi^*(a \mid s, z)$ that rationalize observed trajectories under the entropy-regularized Bellman equation. Estimation proceeds via an EM algorithm: the E-step computes posterior segment membership from trajectory likelihoods, and the M-step updates each segment's reward via AIRL with the anchor constraints applied per segment.

\subsection{IQ-Learn}

The IQ-Learn estimator of \citet{garg2021} uses the inverse Bellman operator $r_Q(s,a) \equiv Q(s,a) - \beta \sum_{s'} P(s' \mid s,a)\, V_Q(s')$, which expresses reward as a function of $Q$ alone. IQ-Learn estimates a single $Q$-function by solving the following optimization problem.
\begin{equation}\label{eq:iq_obj}
\max_Q \;\mathbb{E}_{(s,a) \sim \rho^*}\bigl[\phi\bigl(r_Q(s,a)\bigr)\bigr] - (1-\beta)\,\mathbb{E}_{s_0 \sim \mu_0}\bigl[V_Q(s_0)\bigr],
\end{equation}
The function $\phi(x) = x - x^2/2$ corresponds to $\chi^2$ divergence. At the optimum, $r_Q$ separates immediate reward from continuation values via the Bellman constraint. IQ-Learn requires the transition kernel $P(s' \mid s, a)$ to compute the inverse Bellman operator and uses it in the objective through $V_Q(s')$. It does not require adversarial training.

\subsection{GLADIUS}

The GLADIUS estimator of \citet{kang2025} parameterizes both $Q_\theta(s,a)$ and $\hat{E}_\theta[V(s') \mid s,a]$ with neural networks, trained jointly on observed $(s,a,s')$ transitions via the following loss.
\begin{equation}\label{eq:gladius}
\mathcal{L}(\theta) = -\frac{1}{N}\sum_{i}\log\pi_\theta(a_i \mid s_i)
  + \lambda\,\frac{1}{N}\sum_{i}\bigl(\hat{E}_\theta[V(s_i') \mid s_i,a_i] - V_\theta(s_i')\bigr)^{2},
\end{equation}
where $V_\theta(s') = \log\sum_{a'}\exp Q_\theta(s',a')$. After training, reward is extracted via the same inverse Bellman operator as IQ-Learn. Unlike IQ-Learn, GLADIUS does not require the transition matrix $P$ at estimation time. It learns $\hat{E}[V(s') \mid s, a]$ from empirical next-state observations rather than computing it from known transitions. The MLP architecture shares parameters across states, providing implicit regularization that tabular $Q$ parameterizations lack.

\subsection{NFXP}

The nested fixed point estimator of \citet{rust1987} parameterizes the flow utility $u_\theta(s,a)$, solves the Bellman equation exactly under known transitions $P(s'\mid s,a)$, and maximizes the log-likelihood of observed actions via the implied softmax probabilities. We use a policy/value iteration hybrid to solve the inner fixed point and a BHHH/BFGS outer loop. Under correct specification, NFXP consistently recovers $r^*$ and yields valid Type B counterfactuals. Its trade-off is computational cost.

\section{Counterfactual Taxonomy}
%% ======================================================================

\begin{definition}[Type A: state extrapolation]
The transition kernel $P$ and reward $r$ are unchanged. Only the realized state values shift. The counterfactual CCPs require only evaluating the estimated Q at new state inputs, with no Bellman re-solve.
\end{definition}

\begin{definition}[Type B: environment change]
The reward $r$ is unchanged, but the transition kernel shifts to $\tilde{P}$ or the action set changes. The counterfactual Q must satisfy a new Bellman equation $\tilde{Q} = r + \beta \tilde{P}\tilde{V}$, which requires the structural reward $r$ separated from the old continuation values.
\end{definition}

\begin{proposition}[Type A equivalence]\label{prop:type1}
Under Type A, all methods considered yield identical counterfactual CCPs.
\end{proposition}
\begin{proof}
Each method recovers scores whose within-state action differences match $Q^*$, which can be written as $Q^*(s,a) + c(s)$ for some action-independent $c$. For any shifted state $\tilde{s}$, the softmax $\pi(a \mid \tilde{s}) \propto \exp(Q^*(\tilde{s},a) + c(\tilde{s}))$ cancels $c(\tilde{s})$. \qed
\end{proof}

\begin{proposition}[Type B divergence]\label{prop:type2}
Under Type B, reduced-form and AIRL-without-anchors produce invalid predictions. AIRL with anchors, IQ-Learn, and GLADIUS produce valid predictions when the structural reward is correctly recovered.
\end{proposition}

The reduced-form estimate $Q^* + c(s)$ bundles $r$ with $\beta PV^*$, so under $\tilde{P}$ it cannot disentangle $r$. AIRL without anchors recovers $g^* = r + \text{shaping}$, which is observationally equivalent in the original MDP but produces incorrect behavior under $\tilde{P}$ unless $\delta \equiv 0$. AIRL with anchors recovers $g^* = r^*$ exactly (Theorem~\ref{thm:recovery}). IQ-Learn and GLADIUS separate $r$ via the inverse Bellman operator. The LSW anchors provide what \citet{kalouptsidi2021} call the ``additional restrictions'' needed to identify counterfactuals under transition changes.

\section{Application to LSW (2026)}
%% ======================================================================

LSW study sequential content consumption with three actions (buy, wait, exit) using AIRL under the two anchors and deterministic transitions. Their wait-time validation (Section 6.3) is a Type A counterfactual, so every method that recovers the advantage function would pass it identically. Their wait-for-free (WFF) removal and segment-specific delay policies (Sections 7.1--7.2) are Type B because they change the action-set structure and the MDP that each segment faces. Their content-based paywall (Section 7.3) is also Type B because high-content episodes become paid-only, altering the transition structure episode by episode.

\section{Simulation Study}

\paragraph{Data sources.} Analyses A--C and E--G report population-level results computed from the solved MDP or from closed-form shaping, with no estimation noise. The JSON artifact for these analyses is \path{experiments/identification/results/full_simulation.json}, which also drives the repository figures. Analysis D reports finite-sample outcomes using the estimator implementations in \texttt{econirl}.

We construct a tabular MDP that mimics the LSW book-reading environment. The state encodes episode position, content quality, and wait-time level, giving $15 \times 3 \times 4 = 180$ regular states plus one absorbing state. The three actions are buy, wait, and exit, with deterministic transitions and $\beta = 0.95$. The true reward is $r^*(s,\text{buy}) = u(c) - 2.0$, $r^*(s,\text{wait}) = u(c) - 0.08w$, and $r^*(s,\text{exit}) = 0$, where $u \in \{0.5, 1.5, 3.0\}$. Analyses A through C and E through G are evaluated at the population level with no estimation noise. Analysis D uses finite-sample estimation with the actual IQ-Learn and GLADIUS implementations from the \texttt{econirl} package.

Methods with anchors or Bellman consistency pin the value function offset and recover $r^*$ exactly. The reduced-form estimator converges to $A^*$, which bundles reward with continuation values. These results clarify the role of \texttt{econirl}. The package is not just a collection of estimators. It is a framework for deciding which estimators support which counterfactual claims. For in-sample fit and Type A state extrapolation, many methods can look equally successful because they recover the same within-state choice differences. For Type B counterfactuals that change transitions, delays, or action sets, the package needs methods that recover a structural reward rather than a reduced-form advantage. That is why \texttt{econirl} supports several estimators under one interface and evaluates them not only by fit, but also by reward recovery and counterfactual validity.
