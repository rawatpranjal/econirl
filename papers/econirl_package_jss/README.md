# econirl JSS Submission

Source for the Journal of Statistical Software paper introducing the `econirl` package.

## Compiling

This paper uses the `jss.cls` document class. Before compiling, download the JSS author resources from the journal website and place `jss.cls` and `jsslogo.jpg` in this directory. Then run

```
pdflatex main
bibtex main
pdflatex main
pdflatex main
```

## Replication

Every figure and table in the paper is generated by a script under `code_snippets/` or under the package's `examples/` tree. See `replication.md` for the full mapping.
