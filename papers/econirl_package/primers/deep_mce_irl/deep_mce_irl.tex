\section{Problem}

Linear MCE-IRL \citep{ziebart2010} uses a reward
$R(s,a)=\theta^\top\phi(s,a)$ and solves the maximum causal entropy
control problem under known transitions. Wulfmeier, Ondruska, and Posner
\citep{wulfmeier2015} replace the linear state reward with a neural map
from supplied state features to rewards. The Bellman recursion and
occupancy matching logic stay the same; only the reward parameterization
changes.

The package implementation validated here is \texttt{MCEIRLNeural}. The
primary Shapeshifter validation uses a frozen nonlinear neural reward,
known stochastic transitions, supplied state encodings, and action 0 as a
zero-reward anchor. The gated artifact for neural-reward truth is the
learned raw \texttt{reward\_matrix}, not a projected finite $\theta$. The
raw neural network weights are not treated as finite economic parameters.
When the truth is linear in a supplied feature matrix, the post-fit
projection is reported and parameter gates apply only when that projection
is numerically identifiable. Counterfactual gates apply the same affine IRL
reward normalization convention before adding intervention deltas.

\section{Objective}

\commonnotation

For a neural reward $R_\psi$, the soft control problem is
\begin{equation}
\max_\pi \;
\E_\pi\!\left[
  \sum_{t=0}^{\infty}\beta^t
  \bigl(R_\psi(s_t,a_t) + \sigma \calH(\pi(\cdot \mid s_t))\bigr)
\right].
\end{equation}
Given a current reward surface, the backward pass solves
\begin{align}
Q(s,a) &= R_\psi(s,a) + \beta \sum_{s'} P(s' \mid s,a)V(s'), \\
V(s) &= \sigma \log \sum_a \exp(Q(s,a)/\sigma), \\
\pi(a\mid s) &= \exp((Q(s,a)-V(s))/\sigma).
\end{align}
The forward pass computes the normalized discounted state-action occupancy
$\mu_\pi(s,a)$. The neural update uses the same occupancy mismatch as
linear MCE-IRL:
\begin{equation}
\nabla_\psi L
 =
\sum_{s,a} \bigl(\mu_\pi(s,a)-\mu_D(s,a)\bigr)
\nabla_\psi R_\psi(s,a).
\end{equation}

\section{Identification}

Neural rewards inherit the usual IRL normalization problem. Additive shifts,
positive rescaling, and richer potential-like reward transformations can leave
the induced policy nearly unchanged \citep{ng1999,kim2021}. In action-dependent
DDC cells we therefore anchor action 0 at zero reward. The validation separates
four objects:

\begin{itemize}
  \item raw neural reward matrix: the gated object when the truth is a frozen
  neural reward;
  \item neural weights: diagnostic only, not structural parameters;
  \item projected reward: used for finite linear truths when the projection
  basis is well conditioned;
  \item induced behavior: policy, value, Q, occupancy, and counterfactual
  performance under the recovered reward map.
\end{itemize}

The success claim below is nonlinear reward-map recovery over supplied state
encodings and known transitions. It is not a claim that raw network weights are
uniquely identified. It is also not a claim that the package learns raw
Objectworld-style spatial features with a convolutional FCNN; that would require
a convolutional reward network rather than the current MLP over supplied
encodings.

\section{Algorithm}

\begin{algorithm}[H]
\DontPrintSemicolon
\caption{Deep MCE-IRL with Anchored Reward-Map Validation}
\KwIn{Demonstrations, known transitions $P$, discount $\beta$, state encoder, optional action or absorbing-state anchor}
Compute discounted empirical occupancy $\mu_D(s,a)$ from trajectories\;
Initialize neural reward network $R_\psi$\;
\For{epochs}{
  Compute anchored reward matrix $R_\psi(s,a)$\;
  Solve the soft Bellman equations for $V$, $Q$, and $\pi$\;
  Compute model occupancy $\mu_\pi(s,a)$ from the empirical initial distribution\;
  Update $\psi$ with surrogate loss
  $\sum_{s,a} R_\psi(s,a)(\mu_\pi(s,a)-\mu_D(s,a))$\;
}
If the truth is a finite linear feature model and the projection is identified,
compute $\hat\theta=\argmin_\theta\|R_\psi-\Phi\theta\|^2$\;
\KwOut{Reward matrix, optional projected parameters, policy, value, Q, and counterfactual metrics}
\end{algorithm}
